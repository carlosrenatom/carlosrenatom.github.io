# Portfólio — Carlos Renato

Site estático bilíngue (PT/EN), feito com HTML, CSS e JavaScript.

## Publicação no GitHub Pages

1. Renomeie este repositório para `carlosrenatom.github.io`.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, abra **Settings → Pages** e selecione **Deploy from a branch**: `main` e a pasta `/(root)`.
4. O site ficará disponível em `https://carlosrenatom.github.io`.

## Atualização de conteúdo

Todo o conteúdo editável, inclusive as duas traduções, está centralizado em `script.js` no objeto `content`.

- `projectsData`: projetos exibidos na seção Projetos.
- `timeline`: itens da jornada.
- `skills`: grupos de tecnologias e competências.

A foto da seção Sobre fica em `assets/carlos-renato-workspace.png`.
